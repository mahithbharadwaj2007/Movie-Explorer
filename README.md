# Movie Explorer

Movie Explorer is a simple web application that allows users to search for movies and view detailed information such as title, release date, IMDb rating, genre, actors, and plot using the OMDb API.

## 🚀 Features
- Search movies by name
- View movie details (title, rating, genre, plot)
- Handles invalid search results
- Displays default poster if unavailable

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- OMDb API
